# Minesweeper-Flet
Minesweeper created using python flet package.

## Setting

### Installation Python

Install Python 3.10.4 or Latest

### Create virtual environment

venv command sample

```bash
python -m venv <ENV NAME>
```

### Installation Package

```bash
# for user
pip install .

# for developer
pip install ".[develop]"
```

## Execution

```
python app.py
```
